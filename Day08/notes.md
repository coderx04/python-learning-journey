Calculates a fun love score based on the letters in two names.

### 3. Caesar Cipher
Encrypts and decrypts text using Caesar Cipher.

Features:
- Encryption
- Decryption
- Handles spaces and symbols
- Supports large shift values using modulo (%)

- Lists
- Strings

- Conditional statements
- Algorithm design
