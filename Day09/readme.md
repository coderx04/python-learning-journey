
# Day 09 - Secret Auction

## Project

A console-based secret auction program.

## Features

- Multiple bidders
- Stores bids in a dictionary
- Clears the screen between bidders
- Determines the highest bidder
- Announces the winner

## Concepts Practiced

- Dictionaries
- Loops
- User input
- Conditional statements
- Dictionary methods
- Built-in max() with key parameter
