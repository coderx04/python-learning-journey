# Day 05 - Python Projects

## Concepts Practiced

* Lists
* For loops
* Conditional statements
* Random module
* Basic algorithms

## Projects

### 1. Highest and Lowest Score Finder

Finds the highest and lowest score from a list without using Python's built-in `max()` or `min()` functions.

### 2. Password Generator

Generates a random password based on the number of letters, numbers, and symbols chosen by the user.

## Learning Notes

While rebuilding these projects, I practiced breaking problems into smaller steps before writing code. This helped me understand the logic behind the solutions instead of only memorizing syntax.
