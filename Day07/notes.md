
# Day 07 - Hangman

## Project

A console-based Hangman game built using Python.

## Features

* Random word selection
* Letter guessing
* Lives system
* Hangman visual stages
* Win and lose conditions
* Tracks previously guessed letters

## Concepts Practiced

* While loops
* For loops
* Lists
* Conditional statements
* String manipulation
* State management
* Game logic

## Reflection

This was one of the first larger projects I built. Rebuilding it later showed me the difference between understanding code and designing a program. The biggest challenge was managing previously guessed letters and preserving game state between turns.
